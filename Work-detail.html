Portfolio site

活動のためのポートフォリオサイトを作成しました。
設計をシンプルにし、デザイン性やアニメーションを前面に出していくことで、自分の世界観を強く表現しました。
文字のフォントや、細かい要素の形などにの細部にこだわり、1つの作品として楽しんでいただけるものに仕上げています。

コンセプト
「シンプル、かつ見ていて楽しい」
使用技術
HTML / CSS / JavaScript / Figma
制作期間
企画 / 情報設計 / WF 1週間 デザイン 1週間 コーディング 2週間
デザインプロセス
線・点・四角などの基本的な要素を組み合わせることで、見やすくもあり整ったデザインにしています。また、配色を白と黒の2色で構成することで、モダンな雰囲気を演出しています。
